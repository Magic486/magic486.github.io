​​ 整数转字符 ​​：
数字 + ‘0’ 的正确场景 ​​：仅适用于 ​​ 单个数字（0-9）​​ 转字符
a % 4 的结果是一个 0 到 3 的整数。
‘0’ 的 ASCII 码是 48，所以：
0 + ‘0’ = 48 → ‘0’
1 + ‘0’ = 49 → ‘1’
2 + ‘0’ = 50 → ‘2’
3 + ‘0’ = 51 → ‘3’
这样就将整数 0-3 转换为对应的字符’0’到’3’。
​​ 字符转整数 ​​：
反过来，字符’0’到’3’减去’0’：
‘0’ - ‘0’ = 0
‘1’ - ‘0’ = 1
‘2’ - ‘0’ = 2
‘3’ - ‘0’ = 3
这样就将字符’0’到’3’转换为对应的整数 0-3。

关键点总结
操作 作用 示例
int + ‘0’ 将整数 0-9 转换为对应字符 2 + ‘0’ → ‘2’
char - ‘0’ 将字符’0’-‘9’转换为对应整数 ‘2’ - ‘0’ → 2
std::to_string() 直接转换任意类型为字符串 std::to_string(42) → “42”
